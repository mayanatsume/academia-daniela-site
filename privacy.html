Política de Privacidade

Recolha de Dados

A Academia Daniela Silva garante aos utilizadores deste website o respeito pela sua privacidade, adotando as medidas necessárias para proteger os seus dados pessoais.

O acesso ao website não implica, por si só, o fornecimento de dados pessoais. No entanto, a utilização de formulários ou serviços poderá implicar a disponibilização de dados.

Recolha e Tratamento de Dados

A Academia Daniela Silva é a entidade responsável pela recolha e tratamento dos dados pessoais dos utilizadores.

Os dados recolhidos incluem:

-Nome

-Email

-Número de telefone

Estes dados destinam-se a:

-Responder a pedidos de contacto

-Gestão de marcações e inscrições

-Comunicação com clientes

Os dados são tratados informaticamente e armazenados de forma segura.

Base Legal

O tratamento dos dados é realizado com base no consentimento do utilizador, nos termos do

Regulamento Geral sobre a Proteção de Dados.

https://eur-lex.europa.eu/PT/legal-content/summary/general-data-protection-regulation-gdpr.html

Conservação dos Dados

Mantemos as suas informações apenas pelo tempo estritamente necessário para cumprir as finalidades descritas, obrigações fiscais ou resolução de possíveis litígios.

Direitos do Utilizador

Nos termos da legislação aplicável, o utilizador tem direito a:

-Acesso aos seus dados pessoais

-Retificação ou atualização dos dados

-Eliminação dos dados

-Limitação ou oposição ao tratamento

-Retirada do consentimento

Para exercer os seus direitos, deverá contactar: academiadanielasilva@gmail.com

Partilha de Dados

Os dados pessoais não serão vendidos nem partilhados com terceiros, exceto quando necessário para:

-Cumprimento de obrigações legais

-Prestação de serviços solicitados

Medidas de Segurança

A Academia Daniela Silva adota medidas técnicas e organizativas adequadas para proteger os dados pessoais contra:

-Acesso não autorizado

-Divulgação indevida

-Perda ou destruição
